Questão 1

Vamos usar o método de árvore de recursão para encontrar o limite superior assintótico para a recorrência \(T(n) = 4T(\lfloor n/2 \rfloor) + cn\).

Nível Tamanho do nó Número de nós Tempo por nó Custo do nível
0 \(n\) \(4^0 = 1\) \(c \cdot n\) \(c \cdot n\)
1 \(\frac{n}{2}\) \(4^1 = 4\) \(c \cdot \frac{n}{2}\) \(4 \cdot c \cdot \frac{n}{2} = 2cn\)
2 \(\frac{n}{4}\) \(4^2 = 16\) \(c \cdot \frac{n}{4}\) \(16 \cdot c \cdot \frac{n}{4} = 4cn\)
\(i\) \(\frac{n}{2^i}\) \(4^i\) \(c \cdot \frac{n}{2^i}\) \(4^i \cdot c \cdot \frac{n}{2^i} = 2^i cn\)
\(\log_2 n\) \(1\) \(4^{\log_2 n} = n^2\) \(\Theta(1)\) \(n^{\log_2 4} = n^2\)

Soma dos custos:

\[ T(n) = \sum_{i=0}^{\log_2 n - 1} 2^i cn + n^2 \] \[ \sum_{i=0}^{\log_2 n - 1} 2^i = 2^{\log_2 n} - 1 = n - 1 \] \[ T(n) = cn \sum_{i=0}^{\log_2 n - 1} 2^i + n^2 = cn (n - 1) + n^2 = cn^2 - cn + n^2 \]

Desconsiderando termos de ordem inferior:

\[ T(n) = O(n^2) \]

Portanto, o limite superior assintótico para a recorrência \(T(n) = 4T(\lfloor n/2 \rfloor) + cn\) é \(O(n^2)\).

Questão 2

Agora, usando o método da substituição, vamos verificar o limite superior assintótico para a recorrência \(T(n) = 4T(\lfloor n/2 \rfloor) + cn\), onde c é uma constante positiva, obtido na questão anterior.

Seja \(T(n) = 4T(\lfloor n/2 \rfloor) + cn\), queremos provar que \(T(n) \leq g \cdot n²\), onde \(n \geq n_0\) e \(g\), \(n > 0\).

Hipótese de indução: \(T(k) \leq g \cdot k^2\), para todo \(k\), onde \(n_0 \leq k < n\).

Passo de indução: queremos provar que \(T(n) \leq g \cdot n^2\).

\[ T(n) = 4T(\lfloor n/2 \rfloor) + cn \leq 4g \cdot \left( \frac{n}{2} \right)^2 + cn \] \[ = 4g \cdot \frac{n^2}{4} + cn \] \[ = g \cdot n^2 + cn \]

A hipótese não foi suficiente, precisamos construir uma hipótese mais forte.

Hipótese de indução 2: queremos provar que \(T(k) \leq g \cdot k^2 - j \cdot k\), para todo \(k\), onde \(n_0 \leq k < n\) e \(g, j, n_0 > 0\).

Passo de indução: queremos provar que \(T(n) \leq g \cdot n^2 - j \cdot k\).

\[ T(n) = 4T(\lfloor n/2 \rfloor) + cn \leq 4(g \cdot \left( \frac{n}{2} \right)^2 - j \cdot \frac{n}{2}) + cn \] \[ = 4g \cdot \frac{n^2}{4} - 2j \cdot n + cn \] \[ = g \cdot n^2 - 2j \cdot n + cn \] \[ = g \cdot n^2 - j \cdot n - j \cdot n + cn \]

Daí, temos:

\[ - j \cdot n + cn \leq 0 \] \[ c \leq j \]

Se \(c \leq j\), então \(T(n) \leq g \cdot n^2 - j \cdot n\).

Portanto, o limite superior assintótico para a recorrência \(T(n) = 4T(\lfloor n/2 \rfloor) + cn\) é \(O(n^2)\).

Questão 3

Vamos usar o método mestre para fornecer limites assintóticos restritos para as recorrências a seguir:

  1. \(T(n) = 2T(n/4) + 1\)

    Neste caso, \(a = 2\), \(b = 4\) e \(f(n) = 1\).

    • \(n^{\log_b a} = n^{\log_4 2} = n^{1/2}\)

    Veja que \(f(n) = 1 \leq n^{log_b a - \epsilon} \leq n^{log_4 2 - 1/2} = n^{1/2 - 1/2} = n^0 = 1\)/, para \(\epsilon = 1/2 = 0,5\).

    Ou seja, \(f(n) = O(n^{log_b a - \epsilon})\) e o caso 1 do método mestre se aplica.

    Portanto, \(T(n) = \Theta(\sqrt{n})\).


  2. \(T(n) = 2T(n/4) + n²\)

    Neste caso, \(a = 2\), \(b = 4\) e \(f(n) = n²\).

    • \(n^{\log_b a} = n^{\log_4 2} = n^{1/2} = n^{0,5}\)

    Veja que \(f(n) = n² \geq n^{log_b a + \epsilon} \geq n^{log_4 2 + 1/2} = n^{1/2 + 1/2} = n^1 = n\), para \(\epsilon = 1/2 = 0,5\) e

    \(af(n/b) \leq cf(n)\), onde \(c < 1\), para \(n > n_0\).
    \(2 \cdot \left( \frac{n}{4} \right)^2 \leq c \cdot n^2\),
    \(2 \cdot \frac{n^2}{16} \leq c \cdot n^2\),
    \(\frac{n^2}{8} \leq c \cdot n^2\),
    \(c \geq \frac{1}{8}\).

    Ou seja, \(f(n) = \Omega(n^{log_b a + \epsilon})\) e o caso 3 do método mestre se aplica.

    Portanto, \(T(n) = \Theta(n²)\).


  3. \(T(n) = 2T(n/4) + n\)

    Neste caso, \(a = 2\), \(b = 4\) e \(f(n) = n\).

    • \(n^{\log_b a} = n^{\log_4 2} = n^{1/2} = n^{0,5}\)

    Veja que \(f(n) = n \geq n^{log_b a - \epsilon} \geq n^{log_4 2 - 0,25} = n^{1/2 - 0,25} = n^{0,25} = \sqrt{n}\), para \(\epsilon = 0,25\) e

    \(af(n/b) \leq cf(n)\), onde \(c < 1\), para \(n > n_0\).
    \(2 \cdot \left( \frac{n}{4} \right) \leq c \cdot n\),
    \(2 \cdot \frac{n}{4} \leq c \cdot n\),
    \(\frac{n}{2} \leq c \cdot n\),
    \(c \geq \frac{1}{2}\).

    Ou seja, \(f(n) = \Omega(n^{log_b a - \epsilon})\) e o caso 3 do método mestre se aplica.

    Portanto, \(T(n) = \Theta(n)\).


  4. \(T(n) = 2T(n/4) + \sqrt{n}\)

    Neste caso, \(a = 2\), \(b = 4\) e \(f(n) = \sqrt{n}\).

    • \(n^{\log_b a} = n^{\log_4 2} = n^{1/2} = n^{0,5}\)

    Veja que \(f(n) = n^{1/2} \leq n^{1/2}log^k n \leq n^{1/2}log^0 n = n^{1/2}\), para \(k = 0\).

    Ou seja, \(f(n) = \Theta(n^{log_b a})\) e o caso 2 do método mestre se aplica.

    Portanto, \(T(n) = \Theta(n^{1/2}log^1 n) = \Theta(\sqrt{n}log n)\).
