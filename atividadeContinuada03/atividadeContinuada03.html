    Hipótese Indutiva: Suponha que para todo \(k\) com \(n_0 \leq k < n\), temos \(T(k) \leq c \cdot k^{\log_3 4}\), onde \(c\) e \(n_0\) são constantes positivas.

    Passo Indutivo: Queremos provar que \(T(n) \leq c \cdot n^{\log_3 4}\).
    \[T(n) \leq 4 \cdot c \cdot (n/3)^{\log_3 4} + n\] \[T(n) \leq 4 \cdot c \cdot (n^{\log_3 4} / 3^{\log_3 4}) + n\] \[T(n) \leq 4 \cdot c \cdot n^{\log_3 4} / 4 + n\] \[T(n) \leq c \cdot n^{\log_3 4} + n\] Para que a inequação seja verdadeira, basta que: \[c \cdot n^{\log_3 4} + n \leq c \cdot n^{\log_3 4}\] \[n \leq 0\] Como a inequação não é verdadeira, a prova de substituição falha.

    Subtraindo um Termo de Ordem Mais Baixa: Vamos subtrair um termo de ordem mais baixa para fazer com que a prova de substituição funcione.
    Passo Indutivo: Queremos provar que \(T(n) \leq c \cdot n^{\log_3 4} - x \cdot n\).
    \[T(n) \leq 4 \cdot c \cdot (n/3)^{\log_3 4} + n - x \cdot n / 3\] \[T(n) \leq 4 \cdot c \cdot (n^{\log_3 4} / 3^{\log_3 4}) + n - x \cdot n / 3\] \[T(n) \leq 4 \cdot c \cdot n^{\log_3 4} / 4 + n - x \cdot n / 3\] \[T(n) \leq c \cdot n^{\log_3 4} - x \cdot n / 3 + n\] \[T(n) \leq c \cdot n^{\log_3 4} - x \cdot n / 3 + n\] \[T(n) \leq c \cdot n^{\log_3 4} - x \cdot n \cdot 2 / 3\] Como \(c \cdot n \cdot 2 / 3 \leq x \cdot n\), temos:
    \[T(n) \leq c \cdot n^{\log_3 4} - x \cdot n\] Portanto, \(T(n) = O(n^{\log_3 4})\).
